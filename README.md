# Quizzy

Welcome to **Quizzy**, a fun terminal-based quiz about Bitcoin! Answer 7 questions to test your knowledge and see if you’re a Bitcoin expert.

## Description
Quizzy is a simple Python project where you answer multiple-choice questions about Bitcoin.

## How to Run the Project

### Prerequisites
1. Make sure you have Python 3 installed.
2. Clone this repository:
   ```bash
   git clone https://github.com/Aloyssac/quizzy-Aloys_SACCHI-CDOF4.git
   ```
3. Navigate to the project directory:
   ```bash
   cd quizzy-Aloys_SACCHI-CDOF4
   ```

### Run the Game
Start the quiz with:
```bash
python quizzy.py
```

### How to Play
- You will be presented with 7 multiple-choice questions about Bitcoin.
- For each question, you have 30 seconds to answer.
- If you don't answer within the time limit, the question will be marked as incorrect and the quiz will move to the next question.
- Try to answer all questions correctly to become a Bitcoin expert!
